\# Work Order Tracker



A full-stack web application for creating, updating, searching, filtering, and tracking service/work orders.



\## Tech Stack



\- Angular

\- TypeScript

\- Node.js

\- Express

\- MySQL

\- SQL



\## Features



\- Create work orders

\- View work orders in a table

\- Search by customer, title, or description

\- Filter by status: New, In Progress, Done

\- Edit existing work orders

\- Delete work orders

\- Store data in MySQL

\- Use SQL JOIN queries between customers and work orders
